## 1. 디자인 토큰

### 1.1 컬러 팔레트

| 토큰 이름 | HEX | 용도 |
|-----------|-----|------|
| Accent · Fill | `#dff6fb` | 주 버튼 배경, 활성 세션, 강조 영역 |
| Accent · Ink | `#0e5e78` | 주 버튼 텍스트, 링크, 아이콘 |
| Status (Green) | `#1f8a5b` | 성공 상태, 저장 확인, 불릿 점 |
| Ink | `#18181b` | 제목, 가장 진한 본문 |
| Text | `#3f3f46` | 일반 본문 텍스트 |
| Muted | `#9b9ba2` | 보조 텍스트, 플레이스홀더 |
| Surface | `#ffffff` | 카드, 말풍선(AI), 입력 필드 배경 |
| Page | `#fbfbfa` | 교번 행, 섹션 배경 |
| Border | `#ebebe9` | 카드 보더, 구분선 (`#e6e6e4`, `#eeeeec`, `#f2f2f0` 혼용) |
| Danger | `#c5453b` | 전체 초기화, 위험 동작 버튼 |
| App BG | `#f1f1ef` | 앱 전체 배경 |

| 역할 | 폰트 | 적용 위치 |

| 메타 · 코드 · 라벨 | JetBrains Mono | 화면 ID 배지, 타임스탬프, 시스템 프롬프트 에디터, 모노 라벨 |

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.min.css" />

## 3. SCR-01 로그인

**목적:** 미인증 사용자 차단 및 단일 비밀번호 인증.

### 레이아웃

```
[앱 배경 #f1f1ef, 전체 화면 중앙 정렬]

  ┌─ 브랜드 로고 영역 ─────────────────────────┐
  │  [아리 배지 — #dff6fb / #0e5e78 22×22]     │
  │  SIWOO AI (JetBrains Mono, 600, 0.1em)      │
  └─────────────────────────────────────────────┘

  ┌─ 로그인 카드 (max-width 300px) ────────────┐
  │  border: 1px solid #eaeae8                  │
  │  border-radius: 10px / background: #fff     │
  │  padding: 16px                              │
  │                                             │
  │  접속 비밀번호        PASSWORD (Mono, gray) │
  │  ┌─────────────────────────────────────┐    │
  │  │  (password input, height 34px)      │    │
  │  └─────────────────────────────────────┘    │
  │                                             │
  │  ┌─ 접속 버튼 ──────────────────────────┐   │
  │  │  bg: #dff6fb / text: #0e5e78, 700  │   │
  │  │  height: 36px / border-radius: 8px │   │
  │  └─────────────────────────────────────┘   │
  └─────────────────────────────────────────────┘

  하단: "로컬 저장 · 같은 WiFi에서만 접속 가능"
        (Muted, 12px)
```

### 컴포넌트 상세

| 컴포넌트 | 스펙 |
|----------|------|
| 브랜드 로고 | 아리 마크(22×22, border-radius 6px, bg #dff6fb) + SIWOO AI 워드마크 (JetBrains Mono 600, letter-spacing .1em) |
| 비밀번호 입력 | type="password", height 34px, border 1px #e3e3e1, border-radius 8px, bg #fcfcfb |
| 접속 버튼 | height 36px, bg #dff6fb, color #0e5e78, font-weight 700, border-radius 8px |
| DEMO 힌트 | 개발 시에만 노출 — 기본 비밀번호 `siwoo` 안내. 실제 배포 시 제거 |

### 상태 · 인터랙션

- **오류:** 입력 필드 shake 애니메이션(CSS keyframes) + "비밀번호가 올바르지 않습니다" 텍스트 노출
- **성공:** JWT 토큰 `localStorage` 저장 → `/` 리다이렉트
- **제출:** Enter 키 또는 접속 버튼 클릭 동일 처리

## 4. SCR-02 메인 채팅

**목적:** 핵심 화면. 좌측 세션 목록 + 우측 대화 영역. Claude 스트리밍 응답 실시간 표시.

### 레이아웃 (데스크탑 1024px+)

```
┌─ 상단 네비게이션 바 ───────────────────────────────────────────────┐
│  [아리 배지] SIWOO AI    [채팅] [메모리] [설정]  [● 로컬] [로그아웃] │
└───────────────────────────────────────────────────────────────────┘

┌─ 사이드바 (240px 고정) ─┐  ┌─ 채팅 영역 (flex: 1) ───────────────────┐
│  + 새 대화              │  │                                          │
│  ─────────────────────  │  │  ┌─ 메모리 주입 안내 칩 ──────────────┐  │
│  SESSIONS (Mono, gray)  │  │  │  이전 대화 기억을 불러왔어요 · N개  │  │
│                         │  │  └────────────────────────────────────┘  │
│  ┌─ 세션 항목 (활성) ─┐ │  │                                          │
│  │ ▶ 오늘 대화        │ │  │  [AI] 안녕하세요, 시우씨!               │
│  │   09:00            │ │  │       오늘은 무엇을 도와드릴까요?       │
│  └────────────────────┘ │  │                                          │
│  ┌─ 세션 항목 ─────────┐│  │           [나] 할일 목록 정리해줘       │
│  │   어제 대화         ││  │                                          │
│  │   2026/6/17        ││  │  [AI] 알겠습니다. 정리해 드릴게요 ▌     │
│  └────────────────────┘│  │                                          │
│                         │  ├──────────────────────────────────────────┤
│                         │  │  ┌─ 입력 필드 ──────────────────┐ [전송] │
│                         │  │  │ 메시지 보내기…               │ [종료] │
│                         │  │  └──────────────────────────────┘        │
└─────────────────────────┘  └──────────────────────────────────────────┘
```

### 컴포넌트 상세

| 컴포넌트 | 스펙 |
|----------|------|
| 상단 네비바 | height 52px, border-bottom 1px #ebebe9, bg #fff. 좌: 로고. 중: 탭(채팅/메모리/설정). 우: 로컬연결 상태 칩 + 로그아웃 |
| + 새 대화 | height 28px, bg #dff6fb, color #0e5e78, font-weight 700, border-radius 7px, 전체 폭 |
| 세션 목록 항목 | height 34px, border-radius 7px. 활성: bg #eef9fc, border 1px #cdeef7. 비활성: bg transparent. 호버: bg #f5f5f3 |
| AI 말풍선 | 좌측 정렬, bg #fff, border 1px #ececea, border-radius 9px, padding 7px 10px, max-width 72%, font-size 11px(설계서) → 14px(실구현). 좌측에 '아리' 아바타 배지 |
| 사용자 말풍선 | 우측 정렬, bg #eef9fc, border 1px #cdeef7, 동일 padding/radius |
| 스트리밍 커서 `▌` | 응답 중 깜박임(CSS animation 0.7s blink), 완료 시 제거 |
| 입력 바 | border 1px #e3e3e1, border-radius 9px, bg #fff, padding 6px 7px. 내부: 텍스트에리어(flex:1) + [종료] chip + [전송] icon 버튼 |
| [종료] 버튼 | chip 스타일, bg #fff, border 1px #e6e6e4, color #71717a |
| [전송] 버튼 | 26×26px, border-radius 7px, bg #dff6fb, 아이콘 |
| 메모리 칩 | 상단 채팅 영역에 인라인 칩, bg #f4f4f2, color #6b6b70, JetBrains Mono |

### 상태 · 인터랙션

- **빈 새 대화:** 인사 메시지 + 추천 프롬프트 3개 노출
- **응답 중:** 전송 버튼 비활성화, 스트리밍 커서 `▌` 깜박임
- **메모리 주입:** 새 세션 시작 시 "이전 대화 기억을 불러왔어요 · N개 세션" 칩 표시
- **Enter:** 전송 / **Shift+Enter:** 줄바꿈
- **모바일:** 사이드바 → 햄버거 드로어 전환 (스크림 오버레이 포함)

## 5. SCR-03 세션 종료 다이얼로그

**목적:** [종료] 클릭 시 표시. 요약 저장 여부 확인.

### 레이아웃

```
[배경: 딤(rgba(0,0,0,0.4)) + 블러]

  ┌─ 다이얼로그 카드 (max-width 280px) ────────┐
  │  border: 1px solid #eaeae8                  │
  │  border-radius: 11px / bg: #fff             │
  │  padding: 16px                              │
  │  box-shadow: 0 6px 18px -10px rgba(0,0,0,0.2) │
  │                                             │
  │  [아이콘 박스 32×32, bg #f4f4f2, radius 8] │
  │                                             │
  │  세션 종료  (font-size 15px, weight 700)    │
  │                                             │
  │  이 대화를 종료하고 요약을 메모리에         │
  │  저장할까요? (11px, #71717a, line-h 1.5)   │
  │  (요약 생성에 수 초가 소요됩니다)           │
  │                                             │
  │  ┌─ 저장 후 종료 ──────────────────────┐   │
  │  │  bg #dff6fb / color #0e5e78 / 700  │   │
  │  │  height 32px / radius 8px           │   │
  │  └─────────────────────────────────────┘   │
  │                                             │
  │  ┌─ 저장 안 하고 종료 ─────────────────┐   │
  │  │  border 1px #e6e6e4 / color #3f3f46 │   │
  │  │  height 32px / radius 8px           │   │
  │  └─────────────────────────────────────┘   │
  └─────────────────────────────────────────────┘
  (취소: Escape 키 또는 배경 클릭)
```

### 컴포넌트 상세

| 컴포넌트 | 스펙 |
|----------|------|
| 저장 후 종료 | bg #dff6fb, color #0e5e78, weight 700, height 32px, margin-bottom 6px |
| 저장 안 하고 종료 | border 1px #e6e6e4, color #3f3f46, weight 600, height 32px |
| 취소 | 별도 버튼 없음 — Escape 또는 배경 딤 클릭으로 닫기 |

### 상태 · 인터랙션

- **요약 저장 선택 시:** 로딩 인디케이터 표시 → 완료 후 토스트 "메모리에 저장되었습니다" → 새 세션 시작
- **등장:** scale-in 애니메이션 (transform: scale(0.95)→1, opacity 0→1)
- **배경:** `backdrop-filter: blur(4px)` + 딤 오버레이

## 9. 공통 인터랙션 원칙

- **라이트 테마 고정** — 다크 모드 미지원 (NFR-03)
- **액센트 일관성** — 주요 CTA: Accent Fill + Ink. 성공/확인: Green. 위험: Danger(Red)
- **폰트 분리 원칙** — 본문은 Pretendard, 모든 메타·코드·배지·타임스탬프는 JetBrains Mono
- **파괴적 동작 보호** — 전체 초기화 등 되돌릴 수 없는 동작은 반드시 확인 다이얼로그
- **일반 동작 피드백** — 삭제, 저장 등 단순 동작은 토스트 메시지로 피드백
- **스트리밍 응답** — `fetch()` + `response.body.getReader()`로 SSE 스트림 수신 (POST 요청이므로 `EventSource` 사용 불가). 첫 토큰 2초 이내, 응답 중 입력 버튼 비활성(NFR-04)
- **오류 처리** — API 오류 시 크래시 없이 에러 메시지 채팅 영역 인라인 노출(NFR-07)
- **스크롤** — 새 메시지 추가 시 채팅 영역 자동 스크롤 to bottom
